STM32WB55-Based BLE Gateway for Fleet Management System Modernization

Developed a BLE gateway prototype to modernize an existing fleet management system by introducing smartphone-based driver identification and wireless GPS data exchange. The solution uses an STM32WB55 microcontroller as a communication bridge between a mobile application and the existing fleet management device through BLE and UART interfaces. Custom BLE GATT services were designed for driver authentication and GPS location notification, with BLE security mechanisms implemented using Passkey Entry and Security Manager functionalities.

BLE-Enabled Health Monitoring Embedded Node with Wireless Protocol Analysis

Developed an embedded BLE health monitoring prototype based on an ESP32-C3 microcontroller, integrating heart rate acquisition, LCD visualization, and remote actuator control. Designed custom BLE GATT services to enable smartphone-based interaction using nRF Connect, allowing remote servo motor control and real-time heart rate data monitoring. The project was extended with BLE packet sniffing using an nRF51 dongle and Wireshark to analyze wireless communication frames and understand BLE protocol exchanges.
